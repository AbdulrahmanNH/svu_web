# 🎓 منصة فعاليات الجامعة الافتراضية السورية (SVU Events Platform)

منصة ويب تفاعلية متكاملة تم تطويرها لاستعراض الفعاليات، الورشات التدريبية، والندوات العلمية الخاصة بالجامعة الافتراضية السورية. يركز المشروع على تقديم تجربة مستخدم سلسة وعصرية تدعم اللغتين العربية والإنجليزية بشكل كامل.

---

## ✨ المميزات الرئيسية (Key Features)

* **نظام ثنائي اللغة الذكي (Smart Multilingual):** تبديل فوري بين العربية والإنجليزية مع تغيير اتجاه واجهة المستخدم (RTL/LTR) وتغيير ملفات Bootstrap تلقائياً لضمان دقة المحاذاة.
* **حفظ تفضيلات اللغة (Language Persistence):** يعتمد المشروع على تقنية `localStorage` لتذكر لغة المستخدم المفضلة، مما يمنع عودة الموقع للغة الافتراضية عند التنقل بين الصفحات أو تحديث المتصفح.
* **تصميم متجاوب بالكامل (Fully Responsive):** واجهة متوافقة تماماً مع جميع الأجهزة (الهواتف الذكية، الأجهزة اللوحية، والحواسيب) بفضل نظام الشبكة في Bootstrap 5.
* **نظام البحث والتصفية:** محرك بحث داخلي يسمح للمستخدم بالعثور على الفعاليات بالاسم، بالإضافة إلى تصنيفات ديناميكية (برمجة، ويب، أمن معلومات، إلخ).
* **تجربة مستخدم احترافية:** تضمين تأثيرات بصرية حديثة مثل الـ Hover الاحترافي، تنسيق الأزرار المستديرة، ونظام التنبيهات (Alerts) في نماذج الاتصال.

---

## 🛠 التكنولوجيات المستخدمة (Technologies Used)

* **HTML5:** لبناء الهيكلية الأساسية للصفحات.
* **CSS3:** تنسيقات مخصصة تشمل تأثيرات التدرج اللوني (Gradients) والتصميم الزجاجي (Glassmorphism).
* **Bootstrap 5:** النسختين (العالمية و RTL) لضمان توافق التصميم مع اتجاهات اللغة.
* **Vanilla JavaScript:** لإدارة منطق الترجمة، تخزين البيانات محلياً، ومعالجة العمليات دون الحاجة لمكتبات خارجية ثقيلة.
* **Bootstrap Icons:** لتوفير أيقونات متجهة عالية الجودة.
